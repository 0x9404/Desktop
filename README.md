# Desktop Windows 11



### Viewing network interfaces
- *>* netsh
- *netsh>* interface ipv4
- *netsh interface ipv4>* show interfaces



### Windows OOBE
- Unplug ethernet or disable NIC
- *>* OOBE\\BYPASSNRO

### Find windows product key
- *>* wmic path softwarelicensingservice get OA3xOriginalProductKey
